# GGBoy-Android

A minimal Android frontend for GGBoy-Core, bringing Game Boy and Game Boy Color emulation to your pocket.

## Features

- Game Boy / Game Boy Color Emulation – Play your favorite GB and GBC titles on the go. (At least most should be supported)

- Save States – Quickly save and load your progress at any moment.

- Adjustable Emulation Speed – Speed up gameplay for grinding

- Intuitive Touch Controls – Responsive on-screen buttons that mimic the classic handheld layout.

- Adjustable Volume

- Portrait and Landscape mode

## Getting Started
Clone with submodules using:  
```bash
git clone --recurse-submodules https://github.com/Georg-S/GGBoy-Desktop.git
```
